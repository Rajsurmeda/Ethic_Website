Overview 
This repository contains the source for World Religions, an educational site that presents clear, respectful and balanced information about major world religions. The site is designed for students, teachers and curious readers.
Key goals:
Provide accurate, neutral overviews of religions with references.
Provide interactive timelines, maps, and image galleries.
Mobile-first responsive design so it works on phones (e.g., Redmi) and desktops.
Easy to deploy to GitHub Pages or Netlify with live preview.
